## Overview

A hands-on session where the group decides what to build together using AI. The aim is to demonstrate how AI tools can augment work and boost productivity across functions, all grounded in the responsible, ethical, and transparent use of AI.

## Background

## Concept: "AI a la Carte"

Attendees choose from a menu of practical AI use cases. The group picks one (or more) and builds it live during the session. Cross-functional audience (not just tech), so demos should be accessible and relevant to different roles.

Key principles:
- Practical, not theoretical
- Audience-driven (they choose what to build)
- Responsible, ethical, and transparent AI use
- Show how AI augments (not replaces) work
